# Repo name ->fleet-telemetry-platform
High-scale EV fleet telemetry ingestion and analytics platform using Nest.JS, PostgreSQL, and Docker.
# Project Overview
This project collects, processes, and analyzes real-time telemetry data from electric vehicle fleets.

It helps in:
- Monitoring vehicle performance
- Tracking location
- Analyzing battery health
- Managing fleet data
# Features
- Real-time data ingestion
- RESTful APIs
- PostgreSQL database
- Dockerized services
- Scalable backend

# Tech Stack
- Backend: NestJS (Node.js)
- Database: PostgreSQL
- Container: Docker & Docker Compose

# Prerequisites
- Docker
- Docker Compose
- Git
